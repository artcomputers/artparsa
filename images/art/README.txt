Put your art images in this folder.

Name them like this so they appear automatically:
  art-01.jpg
  art-02.jpg
  ...
  art-30.jpg

Tips:
- Two-digit numbers (01, 02 ... not 1, 2).
- Default extension is .jpg. Using .png or .webp? Change "ext" for this
  section in gallery.js.
- Order on the page follows the numbers.
